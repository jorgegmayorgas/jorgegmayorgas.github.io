    Getting Started

    Prerequisites

    1. First install Python 3.10.11 or above
    2. Install pip sh py https://bootstrap.pypa.io/get-pip.py

    3. Choose one of the following solutions:

    4. Install Jupyter Notebook

      sh pip install notebook

    5. Install Visual Studio Code, click on this link

    Installation

    1. Clone the repo sh git clone https://github.com/jorgegmayorgas/ML_Project_YouTube_keyword_optimization.git
    2. Install Python libraries sh pip install -r installed_packages.txt
